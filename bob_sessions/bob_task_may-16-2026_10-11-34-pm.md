# BCB (Better Call Bob) - Architecture Document

## Overview
BCB is an autonomous CLI tool for auditing and repairing vibe-coded projects. It combines static analysis, AI-powered insights, and automated repair capabilities to transform hastily-written code into production-ready software.

## Core Philosophy
- **Autonomous**: Minimal user intervention required
- **Intelligent**: AI-powered analysis and decision-making
- **Actionable**: Not just reports - actual fixes
- **Extensible**: Plugin architecture for custom rules
- **Language-agnostic**: Support for multiple languages (starting with TypeScript/JavaScript)

## Architecture Layers

### 1. CLI Layer
- **Framework**: Commander.js
- **Responsibilities**: 
  - Command parsing and routing
  - User interaction and prompts
  - Progress reporting
  - Output formatting

### 2. Scanner Layer
- **Components**:
  - File system traversal
  - Git integration
  - Ignore pattern handling (.gitignore, .bcbignore)
  - Project type detection
- **Output**: Project manifest with file metadata

### 3. Analysis Layer
- **Modules**:
  - **Code Quality Analyzer**: ESLint integration, complexity metrics, code smells
  - **Structure Analyzer**: Project organization, naming conventions, file structure
  - **Dependency Analyzer**: Package vulnerabilities, outdated deps, unused deps
  - **Security Analyzer**: Common vulnerabilities, secrets detection
  - **Documentation Analyzer**: Missing docs, outdated comments
  - **Test Coverage Analyzer**: Missing tests, test quality
- **Parser**: AST-based analysis using @babel/parser, typescript compiler API
- **Output**: Structured audit report with severity levels

### 4. AI Integration Layer
- **Providers**: OpenAI, Anthropic (Claude)
- **Capabilities**:
  - Context-aware code analysis
  - Intelligent fix suggestions
  - Code generation for missing components
  - Refactoring recommendations
- **Prompt Engineering**: Specialized prompts for different analysis types

### 5. Repair Engine
- **Strategies**:
  - **Automated**: Safe, deterministic fixes (formatting, imports, simple refactors)
  - **AI-Assisted**: Complex refactors with AI guidance
  - **Interactive**: User-approved changes for critical modifications
- **Safety**: 
  - Git integration for rollback
  - Dry-run mode
  - Backup creation
  - Change preview

### 6. Report Generation
- **Formats**: JSON, HTML, Markdown, Terminal
- **Content**:
  - Executive summary
  - Detailed findings by category
  - Severity distribution
  - Fix recommendations
  - Before/after comparisons

### 7. Configuration System
- **Files**: .bcbrc.json, bcb.config.js
- **Options**:
  - Rule severity overrides
  - Ignored paths
  - AI provider settings
  - Auto-fix preferences
  - Custom rules

## Data Flow

```
User Command
    ↓
CLI Parser
    ↓
Scanner → Project Manifest
    ↓
Analysis Layer → Audit Report
    ↓
AI Integration → Enhanced Insights
    ↓
Repair Engine → Fix Plan
    ↓
User Approval (if interactive)
    ↓
Apply Fixes → Modified Project
    ↓
Report Generation → Final Report
```

### 3. Watch Mode

bcb watch [path]

- Continuous monitoring
- Real-time feedback

### 4. Interactive Mode
```bash
bcb interactive [path]
```
- Guided repair process
- Step-by-step fixes
- Learning mode

### 5. CI/CD Integration
```bash
bcb ci [path]
```
- Exit codes for pipeline integration
- Threshold-based pass/fail
- Artifact generation

import { BCBConfig } from '../types';

export class ConfigLoader {
  private static CONFIG_FILES = [
    '.bcbrc.json',
    '.bcbrc.js',
    'bcb.config.js',
    'bcb.config.json',
  ];

  static async load(projectPath: string): Promise<BCBConfig> {
    // Try to find config file
    for (const configFile of this.CONFIG_FILES) {
      const configPath = path.join(projectPath, configFile);
      
      if (await fs.pathExists(configPath)) {
        try {
          logger.info(`Loading config from ${configFile}`);
          const userConfig = await this.loadConfigFile(configPath);
          return this.mergeConfig(defaultConfig, userConfig);
        } catch (error) {
          logger.warn(`Failed to load config from ${configFile}: ${error}`);
        }
      }
    }

    // Check for config in package.json
    const packageJsonPath = path.join(projectPath, 'package.json');
    if (await fs.pathExists(packageJsonPath)) {
      try {
        const packageJson = await fs.readJson(packageJsonPath);
        if (packageJson.bcb) {
          logger.info('Loading config from package.json');
          return this.mergeConfig(defaultConfig, packageJson.bcb);
        }
      } catch (error) {
        logger.warn(`Failed to load config from package.json: ${error}`);
      }
    }

    // Load from environment variables
    const envConfig = this.loadFromEnv();
    if (Object.keys(envConfig).length > 0) {
      logger.info('Loading config from environment variables');
      return this.mergeConfig(defaultConfig, envConfig);
    }

    logger.info('Using default configuration');
    return defaultConfig;
  }

  private static async loadConfigFile(configPath: string): Promise<Partial<BCBConfig>> {
    const ext = path.extname(configPath);
    
    if (ext === '.json') {
      return await fs.readJson(configPath);
    } else if (ext === '.js') {
      // Dynamic import for .js config files
      const configModule = require(configPath);
      return configModule.default || configModule;
    }
    
    throw new Error(`Unsupported config file format: ${ext}`);
  }

  private static loadFromEnv(): Partial<BCBConfig> {
    const config: Partial<BCBConfig> = {};

    if (process.env.BCB_AI_PROVIDER) {
      config.ai = {
        provider: process.env.BCB_AI_PROVIDER as 'openai' | 'anthropic',
        apiKey: process.env.BCB_AI_API_KEY,
        model: process.env.BCB_AI_MODEL,
        maxTokens: process.env.BCB_AI_MAX_TOKENS ? parseInt(process.env.BCB_AI_MAX_TOKENS) : undefined,
      };

        requireApproval: process.env.BCB_AUTO_FIX_APPROVAL !== 'false',

      };
    }

import fs from 'fs-extra';
import path from 'path';
import { glob } from 'glob';
import ignore from 'ignore';
import simpleGit from 'simple-git';
import { FileMetadata, ProjectManifest } from '../types';
import { logger } from '../utils/logger';

export class ProjectScanner {
  private ignorePatterns: string[] = [];
  private ig: ReturnType<typeof ignore>;

  constructor(ignorePatterns: string[] = []) {
    this.ignorePatterns = ignorePatterns;
    this.ig = ignore().add(ignorePatterns);
  }

  async scan(projectPath: string): Promise<ProjectManifest> {
    logger.info(`Scanning project at ${projectPath}`);

    // Load .gitignore if exists
    await this.loadGitignore(projectPath);

    // Detect project type
    const projectType = await this.detectProjectType(projectPath);

    // Check for Git
    const hasGit = await this.hasGitRepo(projectPath);

    // Scan files
    const files = await this.scanFiles(projectPath);

    // Analyze package.json if exists
    const packageInfo = await this.analyzePackageJson(projectPath);

    // Calculate statistics
    const totalLines = files.reduce((sum, file) => sum + file.lines, 0);
    const languages = this.calculateLanguageStats(files);
    const hasTests = files.some(f => 
      f.relativePath.includes('test') || 
      f.relativePath.includes('spec') ||
      f.relativePath.includes('__tests__')
    );

    const manifest: ProjectManifest = {
      rootPath: projectPath,
      projectType,
      files,
      totalFiles: files.length,
      totalLines,
      languages,
      hasGit,
      hasTests,
      ...packageInfo,
    };

    logger.info(`Scan complete: ${files.length} files, ${totalLines} lines`);
    return manifest;
  }

  private async loadGitignore(projectPath: string): Promise<void> {
    const gitignorePath = path.join(projectPath, '.gitignore');
    if (await fs.pathExists(gitignorePath)) {
      const content = await fs.readFile(gitignorePath, 'utf-8');
      const patterns = content.split('\n').filter(line => line.trim() && !line.startsWith('#'));
      this.ig.add(patterns);
    }

    // Also check for .bcbignore
    const bcbignorePath = path.join(projectPath, '.bcbignore');
    if (await fs.pathExists(bcbignorePath)) {
      const content = await fs.readFile(bcbignorePath, 'utf-8');
      const patterns = content.split('\n').filter(line => line.trim() && !line.startsWith('#'));
      this.ig.add(patterns);
    }
  }

  private async detectProjectType(projectPath: string): Promise<string> {
    const packageJsonPath = path.join(projectPath, 'package.json');
    
    if (await fs.pathExists(packageJsonPath)) {
      const packageJson = await fs.readJson(packageJsonPath);
      
      // Check for framework-specific dependencies
      const deps = { ...packageJson.dependencies, ...packageJson.devDependencies };
      
      if (deps.react || deps['react-dom']) return 'React';
      if (deps.vue || deps['@vue/cli']) return 'Vue';
      if (deps['@angular/core']) return 'Angular';
      if (deps.next) return 'Next.js';
      if (deps.nuxt) return 'Nuxt.js';
      if (deps.express) return 'Express';
      if (deps.nestjs) return 'NestJS';
      
      return 'Node.js';
    }

    if (await fs.pathExists(path.join(projectPath, 'requirements.txt'))) {
      return 'Python';
    }

    if (await fs.pathExists(path.join(projectPath, 'Cargo.toml'))) {
      return 'Rust';
    }

    if (await fs.pathExists(path.join(projectPath, 'go.mod'))) {
      return 'Go';
    }

    return 'Unknown';
  }

  private async hasGitRepo(projectPath: string): Promise<boolean> {
    try {
      const git = simpleGit(projectPath);
      await git.status();
      return true;
    } catch {
      return false;
    }
  }

  private async scanFiles(projectPath: string): Promise<FileMetadata[]> {
    const pattern = '**/*';
    const files = await glob(pattern, {
      cwd: projectPath,
      nodir: true,
      dot: true,
      absolute: false,
    });

    const fileMetadata: FileMetadata[] = [];

    for (const file of files) {
      const relativePath = file;
      
      // Check if file should be ignored
      if (this.ig.ignores(relativePath)) {
        continue;
      }

      const fullPath = path.join(projectPath, file);
      
      try {
        const stats = await fs.stat(fullPath);
        const extension = path.extname(file);
        const language = this.detectLanguage(extension);

        // Count lines for text files
        let lines = 0;
        if (this.isTextFile(extension)) {
          const content = await fs.readFile(fullPath, 'utf-8');
          lines = content.split('\n').length;
        }

        fileMetadata.push({
          path: fullPath,
          relativePath,
          size: stats.size,
          extension,
          language,
          lines,
          lastModified: stats.mtime,
        });
      } catch (error) {
        logger.warn(`Failed to process file ${file}: ${error}`);
      }
    }

    return fileMetadata;
  }

  private async analyzePackageJson(projectPath: string): Promise<{
    packageManager?: 'npm' | 'yarn' | 'pnpm';
    dependencies?: Record<string, string>;
    devDependencies?: Record<string, string>;
  }> {
    const packageJsonPath = path.join(projectPath, 'package.json');
    
    if (!(await fs.pathExists(packageJsonPath))) {
      return {};
    }

    const packageJson = await fs.readJson(packageJsonPath);

    // Detect package manager
    let packageManager: 'npm' | 'yarn' | 'pnpm' | undefined;
    if (await fs.pathExists(path.join(projectPath, 'yarn.lock'))) {
      packageManager = 'yarn';

import { parse } from '@babel/parser';
import traverse from '@babel/traverse';
import * as t from '@babel/types';
import fs from 'fs-extra';
import { FileMetadata } from '../types';
import { logger } from '../utils/logger';

export interface ParsedCode {
  ast: any;
  functions: FunctionInfo[];
  classes: ClassInfo[];
  imports: ImportInfo[];
  exports: ExportInfo[];
  complexity: number;
  issues: CodeIssue[];
}

export interface FunctionInfo {
  name: string;
  line: number;
  params: string[];
  isAsync: boolean;
  complexity: number;
  hasErrorHandling: boolean;
}

export interface ClassInfo {
  name: string;
  line: number;
  methods: string[];
  properties: string[];
}

export interface ImportInfo {
  source: string;
  specifiers: string[];
  line: number;
}

export interface ExportInfo {
  name: string;
  type: 'default' | 'named';
  line: number;
}

export interface CodeIssue {
  type: string;
  message: string;
  line: number;
  column: number;
  severity: 'error' | 'warning' | 'info';
}

export class CodeParser {
  async parseFile(file: FileMetadata): Promise<ParsedCode | null> {
    try {
      const content = await fs.readFile(file.path, 'utf-8');
      return this.parseCode(content, file.extension);
    } catch (error) {
      logger.warn(`Failed to parse ${file.path}: ${error}`);
      return null;
    }
  }

  parseCode(code: string, extension: string): ParsedCode {
    const isTypeScript = extension === '.ts' || extension === '.tsx';
    const isJSX = extension === '.jsx' || extension === '.tsx';

    const plugins: any[] = ['decorators-legacy'];
    if (isTypeScript) plugins.push('typescript');
    if (isJSX) plugins.push('jsx');

    const ast = parse(code, {
      sourceType: 'module',
      plugins,
      errorRecovery: true,
    });

    const functions: FunctionInfo[] = [];
    const classes: ClassInfo[] = [];
    const imports: ImportInfo[] = [];
    const exports: ExportInfo[] = [];
    const issues: CodeIssue[] = [];
    let totalComplexity = 0;

    traverse(ast, {
      FunctionDeclaration: (path) => {
        const func = this.extractFunctionInfo(path.node);
        functions.push(func);
        totalComplexity += func.complexity;
      },
      FunctionExpression: (path) => {
        const func = this.extractFunctionInfo(path.node);
        functions.push(func);
        totalComplexity += func.complexity;
      },
      ArrowFunctionExpression: (path) => {
        const func = this.extractFunctionInfo(path.node);
        functions.push(func);
        totalComplexity += func.complexity;
      },
      ClassDeclaration: (path) => {
        const classInfo = this.extractClassInfo(path.node);
        classes.push(classInfo);
      },
      ImportDeclaration: (path) => {
        const importInfo = this.extractImportInfo(path.node);
        imports.push(importInfo);
      },
      ExportNamedDeclaration: (path) => {
        const exportInfo = this.extractExportInfo(path.node, 'named');
        if (exportInfo) exports.push(exportInfo);
      },
      ExportDefaultDeclaration: (path) => {
        const exportInfo = this.extractExportInfo(path.node, 'default');
        if (exportInfo) exports.push(exportInfo);
      },
      // Detect common issues
      ConsoleCallExpression: (path) => {
        if (
          t.isMemberExpression(path.node.callee) &&
          t.isIdentifier(path.node.callee.object, { name: 'console' })
        ) {
          issues.push({
            type: 'console-statement',
            message: 'Console statement found',
            line: path.node.loc?.start.line || 0,
            column: path.node.loc?.start.column || 0,
            severity: 'warning',
          });
        }
      },
      DebuggerStatement: (path) => {
        issues.push({
          type: 'debugger-statement',
          message: 'Debugger statement found',
          line: path.node.loc?.start.line || 0,
          column: path.node.loc?.start.column || 0,
          severity: 'error',
        });
      },
    });

    return {
      ast,
      functions,
      classes,
      imports,
      exports,
      complexity: totalComplexity,
      issues,
    };
  }

  private extractFunctionInfo(node: any): FunctionInfo {
    const name = node.id?.name || 'anonymous';
    const line = node.loc?.start.line || 0;
    const params = node.params.map((p: any) => {
      if (t.isIdentifier(p)) return p.name;
      if (t.isRestElement(p) && t.isIdentifier(p.argument)) return `...${p.argument.name}`;
      return 'unknown';
    });
    const isAsync = node.async || false;
    const complexity = this.calculateComplexity(node);
    const hasErrorHandling = this.checkErrorHandling(node);

    return {
      name,
      line,
      params,
      isAsync,
      complexity,
      hasErrorHandling,
    };
  }

  private extractClassInfo(node: any): ClassInfo {
    const name = node.id?.name || 'anonymous';
    const line = node.loc?.start.line || 0;
    const methods: string[] = [];
    const properties: string[] = [];

    if (node.body && node.body.body) {
      for (const member of node.body.body) {
        if (t.isClassMethod(member) && t.isIdentifier(member.key)) {
          methods.push(member.key.name);
        } else if (t.isClassProperty(member) && t.isIdentifier(member.key)) {
          properties.push(member.key.name);
        }
      }
    }

    return { name, line, methods, properties };
  }

  private extractImportInfo(node: any): ImportInfo {
    const source = node.source.value;
    const line = node.loc?.start.line || 0;
    const specifiers = node.specifiers.map((spec: any) => {
      if (t.isImportDefaultSpecifier(spec)) return spec.local.name;
      if (t.isImportSpecifier(spec)) return spec.local.name;
      if (t.isImportNamespaceSpecifier(spec)) return `* as ${spec.local.name}`;
      return 'unknown';
    });

    return { source, specifiers, line };
  }

  private extractExportInfo(node: any, type: 'default' | 'named'): ExportInfo | null {
    const line = node.loc?.start.line || 0;

    if (type === 'default') {
      return {
        name: 'default',
        type: 'default',
        line,
      };
    }

    if (node.declaration) {
      if (t.isFunctionDeclaration(node.declaration) && node.declaration.id) {
        return {
          name: node.declaration.id.name,
          type: 'named',
          line,
        };
      }
      if (t.isClassDeclaration(node.declaration) && node.declaration.id) {
        return {
          name: node.declaration.id.name,
          type: 'named',
          line,
        };
      }
      if (t.isVariableDeclaration(node.declaration)) {
        const declarator = node.declaration.declarations[0];
        if (t.isIdentifier(declarator.id)) {
          return {
            name: declarator.id.name,
            type: 'named',
            line,
          };
        }
      }
    }

    return null;
  }

  private calculateComplexity(node: any): number {
    let complexity = 1; // Base complexity

    traverse(node, {
      IfStatement: () => complexity++,
      ConditionalExpression: () => complexity++,
      ForStatement: () => complexity++,
      ForInStatement: () => complexity++,
      ForOfStatement: () => complexity++,
      WhileStatement: () => complexity++,
      DoWhileStatement: () => complexity++,
      SwitchCase: (path) => {
        if (path.node.test) complexity++; // Don't count default case
      },
      CatchClause: () => complexity++,
      LogicalExpression: (path) => {
        if (path.node.operator === '&&' || path.node.operator === '||') {
          complexity++;
        }
      },
    }, node);

    return complexity;
  }

  private checkErrorHandling(node: any): boolean {
    let hasErrorHandling = false;

    traverse(node, {
      TryStatement: () => {
        hasErrorHandling = true;
      },
      CatchClause: () => {
        hasErrorHandling = true;
      },
    }, node);

    return hasErrorHandling;
  }
}

  async analyze(context: AnalysisContext): Promise<Finding[]> {

  private async checkHardcodedSecrets(context: AnalysisContext): Promise<Finding[]> {
    const findings: Finding[] = [];

    if (!context.file || !context.content) {
      return findings;
    }

    const secretPatterns = [
      { pattern: /(?:api[_-]?key|apikey)\s*[:=]\s*['"]([^'"]+)['"]/gi, name: 'API Key' },
      { pattern: /(?:secret[_-]?key|secretkey)\s*[:=]\s*['"]([^'"]+)['"]/gi, name: 'Secret Key' },

      const line = lines[i];

        const matches = line.matchAll(pattern);
        for (const match of matches) {

            fixStrategy: FixStrategy.MANUAL,

          });
        }
      }
    }

    return findings;

import { AuditReport, Finding, Severity, FindingCategory, AnalysisContext, BCBConfig } from '../types';
import { ProjectScanner } from '../scanner/project-scanner';
import { QualityAnalyzer } from '../analyzers/quality-analyzer';
import { SecurityAnalyzer } from '../analyzers/security-analyzer';
import { logger, ConsoleReporter } from '../utils/logger';
import fs from 'fs-extra';

export class AuditEngine {
  private scanner: ProjectScanner;
  private qualityAnalyzer: QualityAnalyzer;
  private securityAnalyzer: SecurityAnalyzer;

  constructor(config: BCBConfig) {
    this.scanner = new ProjectScanner(config.ignore);
    this.qualityAnalyzer = new QualityAnalyzer();
    this.securityAnalyzer = new SecurityAnalyzer();
  }

  async audit(projectPath: string, config: BCBConfig): Promise<AuditReport> {
    logger.info(`Starting audit of ${projectPath}`);
    ConsoleReporter.section('Project Audit');

    // Scan project
    ConsoleReporter.info('Scanning project structure...');
    const manifest = await this.scanner.scan(projectPath);
    ConsoleReporter.success(`Found ${manifest.totalFiles} files (${manifest.totalLines} lines)`);

    // Analyze files
    ConsoleReporter.info('Analyzing code quality and security...');
    const findings: Finding[] = [];
    let processedFiles = 0;

    for (const file of manifest.files) {
      processedFiles++;
      ConsoleReporter.progress(processedFiles, manifest.totalFiles, `Analyzing ${file.relativePath}`);

      const content = await this.readFileContent(file.path);
      const context: AnalysisContext = {
        manifest,
        config,
        file,
        content,
      };

      // Run analyzers
      try {
        const qualityFindings = await this.qualityAnalyzer.analyze(context);
        findings.push(...qualityFindings);

        const securityFindings = await this.securityAnalyzer.analyze(context);
        findings.push(...securityFindings);
      } catch (error) {
        logger.warn(`Failed to analyze ${file.path}: ${error}`);
      }
    }

    ConsoleReporter.success(`Analysis complete: ${findings.length} findings`);

    // Generate summary
    const summary = this.generateSummary(findings);
    const score = this.calculateScore(findings, manifest.totalLines);
    const recommendations = this.generateRecommendations(findings, manifest);

    const report: AuditReport = {
      timestamp: new Date(),
      projectPath,
      manifest,
      findings,
      summary,
      score,
      recommendations,
    };

    logger.info(`Audit complete. Score: ${score}/100`);
    return report;
  }

  private async readFileContent(filePath: string): Promise<string | undefined> {
    try {
      return await fs.readFile(filePath, 'utf-8');
    } catch {
      return undefined;
    }
  }

  private generateSummary(findings: Finding[]) {
    const summary = {
      totalFindings: findings.length,
      bySeverity: {
        [Severity.CRITICAL]: 0,
        [Severity.HIGH]: 0,
        [Severity.MEDIUM]: 0,
        [Severity.LOW]: 0,
        [Severity.INFO]: 0,
      },
      byCategory: {
        [FindingCategory.CODE_QUALITY]: 0,
        [FindingCategory.STRUCTURE]: 0,
        [FindingCategory.SECURITY]: 0,
        [FindingCategory.PERFORMANCE]: 0,
        [FindingCategory.DOCUMENTATION]: 0,
        [FindingCategory.TESTING]: 0,
        [FindingCategory.DEPENDENCIES]: 0,
        [FindingCategory.BEST_PRACTICES]: 0,
      },
      autoFixableCount: 0,
    };

    for (const finding of findings) {
      summary.bySeverity[finding.severity]++;
      summary.byCategory[finding.category]++;
      if (finding.autoFixable) {
        summary.autoFixableCount++;
      }
    }

    return summary;
  }

  private calculateScore(findings: Finding[], totalLines: number): number {
    // Start with perfect score
    let score = 100;

    // Deduct points based on severity
    const severityWeights = {
      [Severity.CRITICAL]: 10,
      [Severity.HIGH]: 5,
      [Severity.MEDIUM]: 2,
      [Severity.LOW]: 0.5,
      [Severity.INFO]: 0.1,
    };

    for (const finding of findings) {
      score -= severityWeights[finding.severity];
    }

    // Normalize by code size (more lenient for larger codebases)
    const sizeMultiplier = Math.min(1, totalLines / 10000);
    score = score + (100 - score) * (1 - sizeMultiplier) * 0.3;

    // Ensure score is between 0 and 100
    return Math.max(0, Math.min(100, Math.round(score)));
  }

  private generateRecommendations(findings: Finding[], manifest: any): string[] {
    const recommendations: string[] = [];

    const summary = this.generateSummary(findings);

    // Critical issues
    if (summary.bySeverity[Severity.CRITICAL] > 0) {
      recommendations.push(
        `🚨 Address ${summary.bySeverity[Severity.CRITICAL]} critical security issues immediately`
      );
    }

    // High priority issues

  private buildHTMLFindings(report: AuditReport): string {
    const bySeverity = this.groupBySeverity(report.findings);
    let html = '';

    for (const severity of [Severity.CRITICAL, Severity.HIGH, Severity.MEDIUM, Severity.LOW, Severity.INFO]) {
      const findings = bySeverity[severity];
      if (findings && findings.length > 0) {
        html += `<h3 class="severity-${severity}">${severity.toUpperCase()} (${findings.length})</h3>`;
        
        for (const finding of findings) {
          html += `
            <div class="finding ${severity}">
                <h4>${finding.title} ${finding.autoFixable ? '<span class="badge auto-fix">Auto-fixable</span>' : ''}</h4>
                <p><strong>Category:</strong> ${finding.category}</p>
                <p><strong>File:</strong> ${finding.file || 'N/A'}${finding.line ? `:${finding.line}` : ''}</p>
                <p>${finding.description}</p>
                ${finding.suggestion ? `<p><strong>Suggestion:</strong> ${finding.suggestion}</p>` : ''}
            </div>
          `;
        }
      }
    }

    return html;
  }

  private groupBySeverity(findings: any[]): Record<Severity, any[]> {
    const grouped: Record<Severity, any[]> = {
      [Severity.CRITICAL]: [],
      [Severity.HIGH]: [],
      [Severity.MEDIUM]: [],
      [Severity.LOW]: [],
      [Severity.INFO]: [],
    };

    for (const finding of findings) {
      grouped[finding.severity].push(finding);
    }

    return grouped;
  }

  private getSeverityIcon(severity: Severity): string {
    const icons = {
      [Severity.CRITICAL]: '🔴',
      [Severity.HIGH]: '🟠',
      [Severity.MEDIUM]: '🟡',
      [Severity.LOW]: '🟢',
      [Severity.INFO]: '🔵',
    };
    return icons[severity] || '⚪';
  }

  private getSeverityWeight(severity: Severity): number {
    const weights = {
      [Severity.CRITICAL]: 5,
      [Severity.HIGH]: 4,
      [Severity.MEDIUM]: 3,
      [Severity.LOW]: 2,
      [Severity.INFO]: 1,
    };
    return weights[severity] || 0;
  }

  private getScoreColor(score: number): string {
    if (score >= 90) return '#28a745';
    if (score >= 70) return '#ffc107';
    if (score >= 50) return '#fd7e14';
    return '#dc3545';
  }

# BCB (Better Call Bob) 🔧

An autonomous CLI tool for auditing and repairing vibe-coded projects. BCB combines static analysis, AI-powered insights, and automated repair capabilities to transform hastily-written code into production-ready software.

## Features

- 🔍 **Comprehensive Code Audit**: Analyzes code quality, security, structure, and best practices
- 🤖 **AI-Powered Analysis**: Leverages GPT-4 or Claude for intelligent code insights
- ⚡ **Automated Repairs**: Fixes common issues automatically with safety checks
- 📊 **Multiple Report Formats**: JSON, HTML, Markdown, and terminal output
- 🔒 **Security Scanning**: Detects hardcoded secrets, vulnerabilities, and insecure dependencies
- 🎯 **CI/CD Integration**: Built-in support for continuous integration pipelines
- 🔧 **Configurable Rules**: Customize severity levels and enable/disable specific checks
- 💾 **Safe Operations**: Git integration and backup creation before modifications

npm install -g bcb-cli

npx bcb-cli audit ./my-project

# Audit current directory
bcb audit

bcb audit ./path/to/project

## Programmatic Usage

```typescript
import { AuditEngine, ConfigLoader, ReportGenerator } from 'bcb-cli';

async function auditProject() {
  const config = await ConfigLoader.load('./my-project');
  const auditEngine = new AuditEngine(config);
  const report = await auditEngine.audit('./my-project', config);
  
  const reportGenerator = new ReportGenerator();
  await reportGenerator.generate(report, 'html', 'report.html');
  
  console.log(`Score: ${report.score}/100`);
  console.log(`Findings: ${report.findings.length}`);
}

auditProject();
```

# Contributing to BCB

Thank you for your interest in contributing to BCB (Better Call Bob)! This document provides guidelines and instructions for contributing.

## Code of Conduct

By participating in this project, you agree to maintain a respectful and inclusive environment for all contributors.

## Getting Started

### Prerequisites

- Node.js 18 or higher
- npm, yarn, or pnpm
- Git

### Development Setup

1. Fork the repository
2. Clone your fork:
   ```bash
   git clone https://github.com/your-username/bcb.git
   cd bcb
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Create a `.env` file:
   ```bash
   cp .env.example .env
   # Add your API keys
   ```

5. Build the project:
   ```bash
   npm run build
   ```

6. Run tests:
   ```bash
   npm test
   ```

## Development Workflow

### Branch Naming

- `feature/` - New features
- `fix/` - Bug fixes
- `docs/` - Documentation updates
- `refactor/` - Code refactoring
- `test/` - Test additions or updates

Example: `feature/add-python-support`

### Making Changes

1. Create a new branch:
   ```bash
   git checkout -b feature/your-feature-name
   ```

2. Make your changes
3. Write or update tests
4. Run tests and linting:
   ```bash
   npm test
   npm run lint
   npm run format
   ```

5. Commit your changes:
   ```bash
   git commit -m "feat: add new feature"
   ```

### Commit Message Convention

We follow the [Conventional Commits](https://www.conventionalcommits.org/) specification:

- `feat:` - New feature
- `fix:` - Bug fix
- `docs:` - Documentation changes
- `style:` - Code style changes (formatting, etc.)
- `refactor:` - Code refactoring
- `test:` - Test additions or updates
- `chore:` - Build process or auxiliary tool changes

Examples:
```
feat: add Python language support
fix: resolve issue with console statement detection
docs: update README with new examples
test: add tests for security analyzer
```

### Pull Request Process

1. Update documentation if needed
2. Add tests for new functionality
3. Ensure all tests pass
4. Update CHANGELOG.md if applicable
5. Push to your fork:
   ```bash
   git push origin feature/your-feature-name
   ```

6. Create a Pull Request with:
   - Clear title and description
   - Reference to related issues
   - Screenshots/examples if applicable
